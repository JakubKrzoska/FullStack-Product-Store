## 📝 Project Description

**FullStack Product Store** is a complete web application designed to handle the core functionalities of an online storefront. It features a decoupled architecture with a robust Node.js backend API that serves product data to a dynamic, responsive frontend. This project demonstrates end-to-end development, from designing relational database tables and writing custom SQL queries, to building out RESTful API endpoints and crafting a modern UI with Tailwind CSS.

## ✨ Key Features

- **Product Catalog:** View a beautifully rendered grid/list of products styled rapidly with Tailwind utility classes.
- **RESTful API Architecture:** Clean and structured backend endpoints to fetch, create, update, and delete product data.
- **Relational Data:** Robust data storage and retrieval using PostgreSQL.
- **Modular Codebase:** Clean separation of concerns with dedicated `frontend` and `backend` directories.

## 🛠️ Technologies Used

### Frontend

- **Library:** React.js
- **Styling:** Tailwind CSS
- **Routing:** React Router

### Backend

- **Runtime Environment:** Node.js
- **Framework:** Express.js
- **Database:** PostgreSQL
- **Data Access:** Raw SQL queries for direct database communication

## 🧠 What I Learned

Building this project was a fantastic hands-on experience in connecting the dots between the client and a relational server. Some of the key takeaways include:

- **Mastering React Hooks:** Gained deep, practical experience managing component state and side effects using `useState` and `useEffect`, as well as handling dynamic routing and navigation with `useParams` and `useNavigate`.
- **Backend Development with Express:** Learned how to leverage the Express.js framework on top of Node.js to build a robust server, handle HTTP requests smoothly, and create clean RESTful API endpoints.
- **Mastering SQL:** Writing custom SQL queries to create tables, insert data, and manage relational database operations directly from the backend.
- **Rapid UI Development:** Leveraging Tailwind CSS to build a responsive and modern user interface without writing massive custom CSS files.
- **API Integration:** Connecting the frontend React interface to the backend server to fetch and display data dynamically.

## 🚀 How It Works & Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine.
- [PostgreSQL](https://www.postgresql.org/) installed and running locally.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/JakubKrzoska/FullStack-Product-Store.git](https://github.com/JakubKrzoska/FullStack-Product-Store.git)
   cd FullStack-Product-Store
   ```
